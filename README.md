# Cambodia Road Accident Risk Prediction System

## English
This project predicts road accident risk in Cambodia using Machine Learning.

Main features:
- Risk prediction: Low, Medium, High
- Cambodia map heatmap
- Province dashboard
- Khmer and English UI
- Safety warning recommendation
- Machine learning models: Logistic Regression, Random Forest, XGBoost

## Dataset Note
This dataset is synthetic/demo data for academic machine learning training. It is inspired by common Cambodia road-safety risk factors such as province, time, weather, vehicle type, road condition, driver behavior, injuries, fatalities, and risk level.
